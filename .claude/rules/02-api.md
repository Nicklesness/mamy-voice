# Правила работы с ElevenLabs API

## Безопасность

- `ELEVENLABS_API_KEY` хранится ТОЛЬКО в env-переменных на сервере
- Все вызовы ElevenLabs проходят через наши API routes (НЕ напрямую с клиента)
- Не логировать API ключи и аудио-данные пользователей

## Endpoints ElevenLabs

- **Instant Voice Cloning**: `POST /v1/voices/add` — загрузка аудиосэмпла
- **Text-to-Speech**: `POST /v1/text-to-speech/{voice_id}` — генерация озвучки
- **Voice status**: `GET /v1/voices/{voice_id}` — проверка голоса

## Обработка ошибок

- Всегда обрабатывать rate limits (429) — показывать пользователю "попробуйте позже"
- Таймауты для генерации — максимум 5 минут polling, потом предложить попробовать снова
- Если клонирование не удалось — понятное сообщение + советы по улучшению записи

## Ограничения MVP

- Один голос на пользователя (перезапись заменяет предыдущий)
- Генерация одной книги за раз
- Нет очереди — если генерация идёт, нужно дождаться
