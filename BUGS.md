# BUGS — Реестр багов

Формат: BUG-XXX, нумерация сквозная. Не удалять исправленные — они остаются как история.

### BUG-001: Next.js не раздаёт runtime файлы из public/
- **Статус**: Fixed
- **Приоритет**: Critical
- **Обнаружен**: 2026-03-13
- **Где**: `src/app/api/generate/route.ts`
- **Воспроизведение**: Сгенерировать озвучку → попытаться воспроизвести
- **Ожидание**: MP3 доступен по URL
- **Реальность**: 404, файл не найден (Next.js production не раздаёт файлы добавленные в public/ после билда)
- **Исправлен**: 2026-03-13 — аудио сохраняется в `data/audio/`, раздаётся через `GET /api/audio?file=xxx.mp3`

### BUG-002: Voice clone принимал только 1 файл вместо 2
- **Статус**: Fixed
- **Приоритет**: High
- **Обнаружен**: 2026-03-13
- **Где**: `src/app/api/voice/clone/route.ts`
- **Воспроизведение**: Записать 2 пассажа → отправить на клонирование
- **Ожидание**: Оба аудиофайла отправляются в ElevenLabs

- **Исправлен**: 2026-03-13 — `formData.getAll()` + `cloneVoice` принимает массив

### BUG-003: MP3 чанки конкатенировались побайтово
- **Статус**: Fixed (workaround)
- **Приоритет**: High
- **Обнаружен**: 2026-03-13
- **Где**: `src/app/api/generate/route.ts`
- **Воспроизведение**: Генерация длинного текста с разбивкой на чанки
- **Ожидание**: Корректный MP3
- **Реальность**: Байтовая конкатенация MP3 файлов ломает формат (каждый чанк имеет свои заголовки)
- **Исправлен**: 2026-03-13 — тексты укорочены до <5000 символов, один запрос без разбивки. Для полных текстов нужно ffmpeg или streaming API.
